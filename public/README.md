# What can I do with it?

Send completely anonymous `messages` or `replies`
- with `attachments`
    - with `spoilers`
- and `custom usernames`
    - that change on demand
- and `custom PFPs`
    - that are `statis`: stays the same
    - that are `dynamic`: changes with username
